# CloudBalance
Built a full-stack FinOps platform using Spring Boot and React Vite with JWT-based
authentication and RBAC for Customer, Admin, and Read-only users, featuring dashboards
for Cost Explorer, Onboarding, Account Management, and AWS Services visualization.
Integrated AWS APIs and FusionCharts to deliver real-time cost analytics and service
utilization insights, enabling organizations to monitor expenses and optimize AWS resource
usage effectively.
